# Nexaguard Website

## Local Setup

### Prerequisites

- [pnpm](https://pnpm.io/) installed globally

### Clone the Repository

Using HTTPS:

```bash
git clone https://github.com/Fletten-IT/Nexaguard-website-v2.git
```

Or using SSH:

```bash
git clone git@github.com:Fletten-IT/Nexaguard-website-v2.git
```

### Install Dependencies

```bash
pnpm install
```

### Prepare the Project

```bash
pnpm prepare
```

## Committing Changes

This project uses [Commitizen](https://github.com/commitizen/cz-cli) for standardized commit messages.

```bash
git add .
git commit
```

Commitizen will guide you through creating a properly formatted commit message.
